ACTION eosdac::dacproposals::runstartwork ( name  proposal_id,

This action is called from a collection of deferred actions which creates the escrow, it will only update status. It will fail if the escrow already exists so a worker can call start work many times if the first fails. This is ony intended to be called as a deferred inline action from this contract therefore will fail without get_self() auth.

Parameters
proposal_idthe ID for the proposal
dac_idthe Dac ID associated with the DAC that this proposa relates to.
